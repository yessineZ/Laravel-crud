SSG : server site generation (generate static page in your page)

ISR : Incremental Static Regeneration (regenerate your static page in your page) 
    - 2 methods to apply ISR :
        1-adding on top of your file : 
             export const revalidate = 3600 // invalidate every hour
        2-const data = await fetch("https://api.vercel.app/blog",{
            next : {revalidate : 3600 } or you can put it 0 every request
        })
        

SSR : Server Site rendering (generate static page in your page)        
        -the server generate the data every single request of the client 
        its slower than ISR 

PPR : partial Prerendering (PPR) 
        combine static and dynamic rendering
